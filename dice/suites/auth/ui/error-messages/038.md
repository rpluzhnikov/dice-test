## Steps

1. Отправить форму регистрации с данными, на которые сервер возвращает код INVALID_USERNAME
   - Expected: Отображается русское сообщение, сопоставленное коду INVALID_USERNAME через codeToRu
2. Отправить форму регистрации с данными, на которые сервер возвращает код WEAK_PASSWORD
   - Expected: Отображается русское сообщение, сопоставленное коду WEAK_PASSWORD через codeToRu
3. Отправить форму регистрации с данными, на которые сервер возвращает код USERNAME_TAKEN
   - Expected: Отображается русское сообщение, сопоставленное коду USERNAME_TAKEN через codeToRu

## Notes

Проверка сопоставления codeToRu: каждый код (INVALID_USERNAME, WEAK_PASSWORD, USERNAME_TAKEN) выводит своё русское сообщение, а не сырой код. Точная орфография текста (nice-tier) вне scope.

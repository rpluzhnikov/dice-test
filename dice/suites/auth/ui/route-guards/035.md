## Steps

1. Без аутентификации перейти на защищённый маршрут /play
   - Expected: Происходит перенаправление на /login
2. Без аутентификации перейти на /login
   - Expected: Страница /login открывается, перенаправления нет
3. Без аутентификации перейти на /register
   - Expected: Страница /register открывается, перенаправления нет
4. Войти с корректными данными и перейти на защищённый маршрут /play
   - Expected: Страница /play отображается, перенаправления на /login нет

## Notes

Таблица решений «маршрут × состояние аутентификации» для направления, заданного спецификацией §6.2: нет токена → защищённый маршрут перенаправляет на /login; публичные /login и /register доступны без токена; при наличии токена защищённый маршрут рендерится.

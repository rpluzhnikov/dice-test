## Steps

1. Отправить POST /api/v1/register с телом {"username":"Bob","password":"Passw0rd1"}
   - Expected: Ответ 201 Created
2. Отправить POST /api/v1/login с телом {"username":"bob","password":"Passw0rd1"} (имя в нижнем регистре)
   - Expected: Ответ 401 Unauthorized
   - Expected: Тело ответа содержит code INVALID_CREDENTIALS

## Notes

Поиск пользователя регистрозависимый (§7): "bob" не совпадает с зарегистрированным "Bob", поэтому вход отклоняется как INVALID_CREDENTIALS (без раскрытия факта существования "Bob").

## Steps

1. Отправить POST /api/v1/login с корректными username и password и получить token
   - Expected: Ответ 200 OK с непустым token
2. Декодировать header токена
   - Expected: header содержит alg со значением HS256
   - Expected: header содержит typ со значением JWT
3. Декодировать payload токена
   - Expected: payload содержит sub, равный userId вошедшего пользователя
   - Expected: payload содержит iat
   - Expected: payload содержит exp
4. Вычислить разницу exp − iat
   - Expected: Разница составляет около 604800 секунд (7 дней) с допуском ±60 секунд

## Notes

needs-tooling: требуется декодирование JWT (база64url header/payload). Срок жизни 7 дней (604800 с) проверяется арифметически по claim'ам, без ожидания реального времени.

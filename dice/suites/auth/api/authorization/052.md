## Steps

1. Отправить POST /api/v1/register с валидным телом и без заголовка Authorization
   - Expected: Ответ не равен 401 Unauthorized
   - Expected: Запрос доходит до обработчика регистрации
2. Отправить POST /api/v1/login с валидным телом и без заголовка Authorization
   - Expected: Ответ не равен 401 Unauthorized
   - Expected: Запрос доходит до обработчика входа

## Notes

Публичные маршруты register и login не требуют токена и не отклоняются middleware при его отсутствии.

## Steps

1. Отправить GET /api/v1/wallet с заголовком Authorization: Bearer not.a.jwt
   - Expected: Ответ 401 Unauthorized
   - Expected: Тело ответа содержит code UNAUTHORIZED
2. Отправить GET /api/v1/wallet с заголовком Authorization: Bearer abcdefghijklmnop (строка без структуры JWT)
   - Expected: Ответ 401 Unauthorized
   - Expected: Тело ответа содержит code UNAUTHORIZED

## Notes

needs-tooling: токен формируется вручную как заведомо нечитаемая строка. Middleware не должен принимать значение, не являющееся валидным JWT.

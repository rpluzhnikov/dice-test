## Steps

1. Отправить POST /api/v1/register с телом {"username":"","password":"abcdef12"} (пустой username)
   - Expected: Ответ 400 Bad Request
   - Expected: Тело ответа содержит code INVALID_USERNAME
   - Expected: Пользователь не создан

## Notes

Пустая строка не входит в допустимый диапазон длины 3..20, поэтому отклоняется как INVALID_USERNAME.

## Steps

1. Отправить POST /api/v1/register с заголовком Content-Type: application/json и некорректным JSON в теле {"username":"abc","password": (синтаксически невалидный JSON)
   - Expected: Ответ 400 Bad Request
   - Expected: Тело ответа содержит code INVALID_BODY
2. Отправить POST /api/v1/register с пустым телом запроса
   - Expected: Ответ 400 Bad Request
   - Expected: Тело ответа содержит code INVALID_BODY

## Notes

Тело, которое невозможно разобрать как JSON-объект (синтаксическая ошибка или отсутствие тела), отклоняется на этапе разбора с кодом INVALID_BODY раньше, чем выполняется валидация полей (precedence §3.2: INVALID_BODY > INVALID_USERNAME > WEAK_PASSWORD).

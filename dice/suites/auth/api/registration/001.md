## Steps

1. Отправить POST /api/v1/register с телом {"username":"abc","password":"abcdef12"} (username длиной 3)
   - Expected: Ответ 201 Created
   - Expected: Тело ответа содержит поле userId
2. Отправить POST /api/v1/register с телом {"username":"abcdefghij1234567890","password":"abcdef12"} (username длиной 20)
   - Expected: Ответ 201 Created
   - Expected: Тело ответа содержит поле userId
3. Отправить POST /api/v1/register с телом {"username":"ab","password":"abcdef12"} (username длиной 2)
   - Expected: Ответ 400 Bad Request
   - Expected: Тело ответа содержит code INVALID_USERNAME
4. Отправить POST /api/v1/register с телом {"username":"abcdefghij12345678901","password":"abcdef12"} (username длиной 21)
   - Expected: Ответ 400 Bad Request
   - Expected: Тело ответа содержит code INVALID_USERNAME

## Notes

BVA по длине username: допустимый диапазон 3..20. Граничные значения: 3 и 20 (валидные), 2 и 21 (невалидные).

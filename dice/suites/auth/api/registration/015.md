## Steps

1. Отправить POST /api/v1/register с телом {"username":"chipsuser","password":"abcdef12"} (валидный запрос)
   - Expected: Ответ 201 Created
   - Expected: Тело ответа содержит поле userId
2. Залогиниться этим пользователем через POST /api/v1/login с тем же username и password
   - Expected: Ответ 200 OK
   - Expected: Тело ответа содержит токен для авторизации
3. Запросить GET /wallet с токеном из шага 2 в заголовке Authorization
   - Expected: Ответ 200 OK
   - Expected: Баланс равен 1000 фишек (STARTING_CHIPS)

## Notes

Сквозная проверка побочного эффекта регистрации: новому пользователю начисляется стартовый баланс STARTING_CHIPS = 1000. Кросс-эндпоинтная цепочка (register → login → wallet) намеренная, так как баланс не возвращается ответом регистрации.

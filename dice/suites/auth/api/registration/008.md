## Steps

1. Отправить POST /api/v1/register с телом {"username":"user_longpw","password":"<98 символов 'a' + '1b'>"} (password длиной 100, есть буква и цифра)
   - Expected: Ответ 201 Created
   - Expected: Тело ответа содержит поле userId

## Notes

Верхний предел длины password в спецификации не задан. Шаг подтверждает, что длинный (100 символов) корректный password принимается. Использованное значение: 98 символов «a», затем «1b» (всего 100 символов, содержит букву и цифру).

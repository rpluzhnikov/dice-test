## Steps

1. Отправить POST /api/v1/register с телом {"username":"user_p8a","password":"abcdef12"} (password длиной 8, есть буква и цифра)
   - Expected: Ответ 201 Created
   - Expected: Тело ответа содержит поле userId
2. Отправить POST /api/v1/register с телом {"username":"user_p7a","password":"abcde12"} (password длиной 7, есть буква и цифра)
   - Expected: Ответ 400 Bad Request
   - Expected: Тело ответа содержит code WEAK_PASSWORD

## Notes

BVA по минимальной длине password: минимум 8. Граничные значения 8 (валидно) и 7 (невалидно), оба содержат обязательные классы (буква + цифра), чтобы изолировать проверку длины.

## Steps

1. Отправить POST /api/v1/register с телом {"username":"user_nodig","password":"abcdefgh"} (длина 8, нет цифры)
   - Expected: Ответ 400 Bad Request
   - Expected: Тело ответа содержит code WEAK_PASSWORD
2. Отправить POST /api/v1/register с телом {"username":"user_nolet","password":"12345678"} (длина 8, нет буквы)
   - Expected: Ответ 400 Bad Request
   - Expected: Тело ответа содержит code WEAK_PASSWORD

## Notes

Password обязан содержать хотя бы одну букву и хотя бы одну цифру. Оба шага длиной 8, чтобы изолировать проверку класса символов от проверки длины.

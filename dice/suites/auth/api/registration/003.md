## Steps

1. Отправить POST /api/v1/register с телом {"username":"Aa9_z","password":"abcdef12"} (буквы верхнего и нижнего регистра, цифра, подчёркивание)
   - Expected: Ответ 201 Created
   - Expected: Тело ответа содержит поле userId

## Notes

Положительная проверка набора разрешённых символов username: латиница A–Z/a–z, цифры 0–9, подчёркивание. Длина 5 входит в диапазон 3..20.

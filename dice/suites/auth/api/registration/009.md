## Steps

1. Отправить POST /api/v1/register с телом {"username":"user_nonascii","password":"Ünïcödé1"} (длина 8, цифра есть, но «буквы» только не-ASCII)
   - Expected: Ответ 400 Bad Request
   - Expected: Тело ответа содержит code WEAK_PASSWORD

## Notes

Производное от §3.1 спецификации: «буква» в требованиях к password означает ASCII a–z/A–Z. Не-ASCII символы (Ü, ï, ö, é) не засчитываются как буква, поэтому требование «хотя бы одна буква» не выполнено и password отклоняется как WEAK_PASSWORD.

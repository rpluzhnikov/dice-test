## Steps

1. Отправить POST /api/v1/register с телом {"username":"user_emptypw","password":""} (пустой password)
   - Expected: Ответ 400 Bad Request
   - Expected: Тело ответа содержит code WEAK_PASSWORD
   - Expected: Пользователь не создан

## Notes

Пустой password не удовлетворяет минимальной длине 8 и обязательным классам символов, поэтому отклоняется как WEAK_PASSWORD.

## Steps

1. Отправить POST /api/v1/register с телом {"username":"user_punctpw","password":"a b 1!!!"} (длина 8, есть буква и цифра, присутствуют пробелы и пунктуация)
   - Expected: Ответ 201 Created
   - Expected: Тело ответа содержит поле userId

## Notes

На password не наложено ограничение по набору символов (в отличие от username). При выполненных требованиях длины (>=8), наличия буквы и цифры пробелы и пунктуация допустимы.

# dice

This project is managed by **[Kensa](https://github.com/your-org/kensa)** — a desktop IDE for QA test cases.

## Getting Started

1. Open this directory in Kensa to browse and edit test cases.
2. Test suites live under `suites/`.
3. Custom fields and project settings are in `.tms/`.

## Structure

| Path | Description |
|------|-------------|
| `suites/` | Test suite directories containing `.md` case files |
| `.tms/schema.yaml` | Custom field definitions |
| `.tms/config.yaml` | Project configuration |
| `.tms/attachments/` | Case attachment files |
| `.tms/trash/` | Soft-deleted cases |
| `CLAUDE.md` | Instructions for AI agents working with this project |
